# HoverCard

O **HoverCard** é um componente de interface utilizado dentro do design system para compor interfaces consistentes.

## Como funciona

Este componente encapsula estilos e comportamento reutilizável.

## Estrutura básica

```tsx
import { HoverCard } from "@/components/ui/hover-card"

export function Example() {
  return (
    <HoverCard>
      Conteúdo
    </HoverCard>
  )
}
```
