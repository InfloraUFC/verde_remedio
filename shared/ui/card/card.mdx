# Card

O **Card** é um componente de interface utilizado dentro do design system para compor interfaces consistentes.

## Como funciona

Este componente encapsula estilos e comportamento reutilizável.

## Estrutura básica

```tsx
import { Card } from "@/components/ui/card"

export function Example() {
  return (
    <Card>
      Conteúdo
    </Card>
  )
}
```
