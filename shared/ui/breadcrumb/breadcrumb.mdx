# Breadcrumb

O **Breadcrumb** é um componente de interface utilizado dentro do design system para compor interfaces consistentes.

## Como funciona

Este componente encapsula estilos e comportamento reutilizável.

## Estrutura básica

```tsx
import { Breadcrumb } from "@/components/ui/breadcrumb"

export function Example() {
  return (
    <Breadcrumb>
      Conteúdo
    </Breadcrumb>
  )
}
```
