# Dialog

O **Dialog** é um componente de interface utilizado dentro do design system para compor interfaces consistentes.

## Como funciona

Este componente encapsula estilos e comportamento reutilizável.

## Estrutura básica

```tsx
import { Dialog } from "@/components/ui/dialog"

export function Example() {
  return (
    <Dialog>
      Conteúdo
    </Dialog>
  )
}
```
