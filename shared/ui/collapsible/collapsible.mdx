# Collapsible

O **Collapsible** é um componente de interface utilizado dentro do design system para compor interfaces consistentes.

## Como funciona

Este componente encapsula estilos e comportamento reutilizável.

```tsx

export function Example() {
  return (
    <Collapsible>
      Conteúdo

  )
}
```
