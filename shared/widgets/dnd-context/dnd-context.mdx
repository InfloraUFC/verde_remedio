# DndProvider

O **DndProvider** encapsula o `DndContext` do **dnd-kit** e fornece uma interface simplificada para manipular eventos de drag-and-drop.

Ele centraliza handlers como:

- `onDragStart`
- `onDragMove`
- `onDragOver`
- `onDragEnd`
- `onDragCancel`

## Uso básico

```tsx
  import { DndProvider } from "@/shared/widgets/dnd-context"

  <DndProvider
    onDragStart={(event) => console.log("start", event)}
    onDragEnd={(event) => console.log("end", event)}
  >
    {children}
  </DndProvider>
```

## Handlers disponíveis
- onDragStart	Executa quando o drag inicia
- onDragMove	Executa durante o movimento
- onDragOver	Executa ao passar sobre outro elemento
- onDragEnd	Executa ao soltar o item
- onDragCancel	Executa se o drag for cancelado


## Tecnologias utilizadas
- dnd-kit
