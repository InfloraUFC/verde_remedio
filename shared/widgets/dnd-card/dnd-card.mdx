# SortableCard

O **SortableCard** é um componente utilizado para criar elementos arrastáveis dentro de listas ordenáveis usando **dnd-kit**.

Ele encapsula a lógica de `useSortable` e permite aplicar animações com **framer-motion**.

## Uso básico

```tsx
import { SortableCard } from "@/shared/widgets/dnd-card"

<SortableCard id="1">
  <Card>
    Conteúdo do card
  </Card>
</SortableCard>
```

## Integração com DndContext

Para funcionar corretamente o componente precisa estar dentro de:
- DndContext
- SortableContext

```tsx
  import { DndContext } from "@dnd-kit/core"
  import { SortableContext } from "@dnd-kit/sortable"
  import { SortableCard } from "@/shared/widgets/dnd-card"

  <DndContext>
    <SortableContext items={items}>
      {items.map(item => (
        <SortableCard key={item.id} id={item.id}>
          {item.content}
        </SortableCard>
      ))}
    </SortableContext>
  </DndContext>
```
